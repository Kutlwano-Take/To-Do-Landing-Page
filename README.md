## Features
- Add new tasks
- Mark tasks as complete (with strikethrough)
- Delete tasks
- Data persists using `localStorage` (tasks stay after refresh)
- Fully responsive (mobile + desktop)
- Minimal and beautiful design

## Tech Stack
- HTML5
- CSS3 (Flexbox & custom animations)
- Vanilla JavaScript (ES6+)
- localStorage for persistence
